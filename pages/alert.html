Alert

An alert displays a short, important message in a way that attracts the user's attention without interrupting the user's task.

Alert box with description

You can use description in Alert boxes. You need to include class alert-description and for buttons add alert-buttonsas shown below.

Do you want to Exit?

Please confirm that you really want to exit from this wonderful app. Also note that your current progress will be lost and you will be reset.

Alert box with title

You can use an alert box only with title. You need to include class alert-heading and for buttons add alert-buttonsas shown below.
